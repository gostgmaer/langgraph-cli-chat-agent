## Checkpointer

Using

```
AsyncSqliteSaver
```

Important discovery

```
AsyncSqliteSaver.from_conn_string(...)
```

returns

```
AsyncGeneratorContextManager
```

NOT

```
AsyncSqliteSaver
```

Correct implementation

```python
context = AsyncSqliteSaver.from_conn_string(...)

checkpointer = await context.__aenter__()

await checkpointer.setup()
```

Shutdown

```python
await context.__aexit__(...)
```

# Key Architectural Decisions

- Use `AsyncSqliteSaver` for LangGraph checkpointing.
- Use `session.id` as the LangGraph `thread_id`.
- Keep the graph node implementation simple with `await llm.ainvoke(...)`; let LangGraph emit streaming events.
- Expose `stream_chat()` as the primary chat API for the CLI.
- Keep provider-specific streaming parsing in a dedicated parser module.
- Use dependency injection instead of global singletons for services like the renderer.
