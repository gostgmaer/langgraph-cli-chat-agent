# LangGraph CLI Chat Agent

> A production-ready, multi-model CLI Chat Agent built with LangGraph, LangChain, and RAG support.

## Features
- Multi-model LLM support (OpenAI, Anthropic, Google)
- LangGraph-powered stateful conversation graph
- Persistent memory with checkpointing & session summarization
- Pluggable tool system (weather, news, search, calculator)
- Full RAG pipeline (PDF, Markdown, Web, CSV loaders)
- Multiple vector store backends (Chroma, FAISS, pgvector, Qdrant)
- Rich CLI interface with streaming output
- SQLite & PostgreSQL storage backends
